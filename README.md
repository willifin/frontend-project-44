### Проект "Игры разума" представляет собой серию из 5 игр в консоли, предполагающий следующую логику - вопрос-ответ.

### Минимальные требования запуска игр: операционная система семейства Linux или ОС Windows с установленной Windows Subsystem for Linux (WSL).

### Установите Node.js последней версии в систему глобально. Скопировать репозиторий проекта можно по ссылке - https://github.com/willifin/frontend-project-44.git Запуск игр реализуется путем ввода команды, которые находятся в файла Makefile, находящимся в корне каталога.
